使用方法:引入js还有创建一个canvas标签并且加上id，如果有多个canvas标签id不能相同, 参数有：canvas对象(ctx),x轴坐标(x),y轴坐标(y),宽度(w),高度(h),颜色填充(fillStyle),边框颜色(strokeStyle),
透明度(opacity),圆的半径(r), 起始弧度(startAngle),结束弧度(endAngle),画圆方向(anticlockwise),x轴位移(translateX),y轴位移(translateY),x轴缩放(scaleX), y轴缩放(scaleY),旋转(rotate),
判断画圆是否要闭合(moveTo),图片路径(src), 开始剪切的 x 坐标位置(sx),开始剪切的 y 坐标位置(sy), 被剪切图像的宽度(sw),被剪切图像的高度(sh)

例如： <canvas id="canvas"></canvas> js地址：
https://github.com/lvguoyi/lvguoyi.github.io/blob/master/Canvas%E6%8F%92%E4%BB%B6%E5%B0%81%E8%A3%85/index.js
使用： var ctx = Rect.canvas({
idName: 'canvas'
})
var aa = new Rects({
canvas: ctx,
x: 100,
y: 100,
w: 100,
h: 100
})

aa.rect() 调用画矩形的方法
aa.arcs() 调用画圆的方法
aa.image() 调用画图片的方法
